# ProjectBoard

A project management application built with **PHP 8**, **MariaDB**, **Bootstrap 5**, and a custom **MVC architecture**.

## 🚀 Features

- Create, edit and delete projects
- Project version management
- Version history with timeline
- Dashboard with project statistics
- Personal and client project management
- Responsive Bootstrap interface

## 🛠️ Technologies

- PHP 8
- MariaDB
- PDO
- MVC Architecture
- Bootstrap 5
- HTML5
- CSS3
- JavaScript

## 📈 Future Plans

- User authentication
- File attachments
- Advanced filtering and search
- Project categories
- REST API
